Objectifs du Projet

Créer une carte mondiale fluide et interactive permettant de consulter en un coup d'œil les données météo et les informations clés de chaque pays.

Ambition

Offrir une expérience cartographique intuitive où chaque interaction (survol ou clic) révèle des données complètes sur le pays : météo actuelle, drapeau, population, superficie et langues parlées.

Contrainte Technique

Limiter les appels API pour éviter les surcharges et garantir une expérience fluide, même sur des interactions rapides — résolu via cache local et debounce sur les événements de survol.

Architecture Technique

1
Carte Leaflet

Initialisation de la carte mondiale avec fond OpenStreetMap et couche GeoJSON des pays

2
REST Countries

Récupération des données pays : drapeau, population, superficie, langues officielles

3
OpenWeatherMap

Appel API avec coordonnées de la capitale pour obtenir température, conditions et vent

4
Événements & Popup

Gestion hover/clic avec affichage d'un popup enrichi sur chaque pays sélectionné

5
Optimisations

Cache local pour éviter les appels dupliqués, debounce pour limiter les requêtes au survol

Sources de Données

🗺️
Leaflet.js

Librairie cartographique open-source. Rendu de la carte mondiale, couches GeoJSON, interaction par pays (hover + clic) et gestion des événements utilisateur.

🌍
REST Countries API

API publique fournissant les données statiques de chaque pays : drapeau SVG, nom officiel, population, superficie totale et liste des langues officielles.

☁️
OpenWeatherMap API

API météo temps réel (endpoint /weather) requêtée avec les coordonnées de la capitale du pays. Retourne température, description météo, vitesse et orientation du vent.

Popup Interactif

À chaque survol ou clic sur un pays, un popup enrichi s'affiche avec l'ensemble des données disponibles — météo temps réel incluse.

  • Drapeau du pays en grand format
  • Nom officiel du pays
  • Population et superficie (km²)
  • Langues officielles parlées
  • Température actuelle (°C)
  • Conditions météo (description + icône)
  • Vitesse et orientation du vent

Points Forts du Projet

🗺️
Interactivité Complète
  • Survol pour aperçu rapide du pays
  • Clic pour popup météo détaillé
  • Mise en surbrillance du pays actif
  • Zoom et navigation fluides
⚡
Performances Optimisées
  • Cache local : données mémorisées, pas de double appel API
  • Debounce sur les survols rapides
  • Chargement asynchrone des données
  • Fallback si API indisponible
🧩
Stack Légère & Déployable
  • Aucun framework lourd requis
  • HTML + CSS + JavaScript vanilla
  • APIs publiques gratuites
  • Déployable sur n'importe quel hébergeur statique

Résultats

Couverture mondiale

Carte fluide couvrant 195+ pays avec mise en évidence visuelle au survol et données enrichies disponibles pour chaque territoire.

Données pays complètes

Affichage systématique du drapeau, des langues, de la population et de la superficie via REST Countries API — données exhaustives et fiables.

Météo temps réel

Conditions météo actualisées en temps réel pour chaque pays : température, description illustrée par icône, vitesse et direction du vent.

Performances maîtrisées

Grâce au cache et au debounce, les requêtes API sont réduites au minimum — expérience fluide même lors d'interactions rapides sur la carte.
